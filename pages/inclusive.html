Language

English is a very widely spoken language, but it's not the only one! Internationalisation (also known as i18n) is the practice of increasing the accessibility of your website by providing translation and country specific date, currency and unit options. Translation can be achieved with JavaScript libraries or tools offered by external API's such as google translate. At a very minimum web developers should write clear semantic html which will allow browser extensions or translation tools to work properly.
